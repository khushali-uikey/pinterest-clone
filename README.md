Pinterest Clone (HTML, CSS, JavaScript)

A simple front-end clone of the Pinterest homepage created using **HTML, CSS, and JavaScript**.
This project replicates the basic layout of Pinterest including the search bar, category tabs, and masonry-style image grid.

📌 Project Overview

The goal of this project is to understand how modern image-based platforms like Pinterest display content in a visually appealing way.
This clone demonstrates how to build a **responsive image grid layout** and simple UI interactions using basic web technologies.

🚀 Features

* Pinterest-style homepage layout
* Search bar UI
* Horizontal category navigation
* Masonry image grid layout
* Responsive design
* Interactive category selection using JavaScript

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

🎯 Learning Purpose

This project was created for learning and practice purposes to understand:

* Web layout design
* Masonry style image grids
* Responsive UI development
* Basic JavaScript interactivity
